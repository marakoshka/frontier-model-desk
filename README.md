Установка
Перейти в папку проекта:
cd frontier-model-desk
Установить зависимости:
npm install
Или в PowerShell:
npm.cmd install
Запуск локально
npm run dev
Или в PowerShell:
npm.cmd run dev
После запуска Vite покажет локальный адрес. Обычно это:
http://localhost:5173/
Открыть этот адрес в браузере.
Сборка проекта
npm run build
Или:
npm.cmd run build
