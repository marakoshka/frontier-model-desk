# Frontier Model Desk

Одностраничный MVP для кейса 10. Сервис превращает аналитику из кейса 8 в инструмент принятия решения: пользователь меняет веса факторов и видит, кто выходит лидером среди frontier AI-компаний.

## Что делает сервис

На одном экране есть слайдеры весов, динамический рейтинг компаний, bar chart, карточка текущего лидера с объяснением и кнопка Export memo.

## Компании в MVP

OpenAI, Google DeepMind / Gemini, Anthropic, xAI / Grok.

## Логика скоринга

У каждой компании есть оценки по пяти критериям: reasoning, distribution, enterprise, economics, governance.

Пользователь меняет веса через слайдеры. Веса автоматически нормализуются, после чего считается итоговый weighted score.
